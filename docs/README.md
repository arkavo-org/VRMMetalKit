# VRMMetalKit Engineering Documentation

User-facing API documentation lives in DocC at
**https://arkavo-org.github.io/VRMMetalKit/**, or build it locally with
`make docs`.

The files in this directory are **engineering reference**: architecture
decision records, performance reports, internal specs, and contributor
guides. They are not curated for end-user consumption and may assume
maintainer-level context.

## Contents

- `adr/` — Architecture Decision Records
- `ARKitIntegration.md`, `SHADERS.md`, `SpringBonePhysicsGuide.md`, etc. — subsystem deep-dives
- `*_REPORT.md` — performance and test-coverage snapshots
- `*_PLAN.md` — internal implementation plans
- `superpowers/` — design specs and implementation plans for in-flight work
